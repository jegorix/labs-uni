Рекомендации по построению блок-схем

Общие правила:
- Овал — блоки начала/конца, подписываем «Старт» и «Финиш».
- Прямоугольник — действия/процедуры; внутри пишем короткий оператор или вызов функции.
- Ромб — условие, внутри указываем проверку вида `if (...)`.
- Параллелограмм — ввод/вывод; текст вида `cin >> ...` или `cout << ...`.
- Стрелки помечаем «да»/«нет» для ответвлений; возвращающиеся стрелки рисуем справа от цикла.

Алгоритм 1. Загрузка данных
Блок-1 овал: `Старт`
Блок-2 процесс: `employeesPath = combinePath(dir,"employees.txt")` и аналогично для остальных файлов
Блок-3 процесс: `clearContainers(); resetIds();`
Блок-4 процесс: `loadPositions();`
Блок-5 процесс: `loadDepartments();`
Блок-6 процесс: `loadEmployees();`
Блок-7 процесс: `loadHireDates();`
Блок-8 овал: `Готово`

Алгоритм 2. Добавление работника
Блок-1 овал: `Старт`
Блок-2 ввод: `read firstName, lastName, departmentId, positionId, salary, bonus, hireDate`
Блок-3 условный: `if (!departmentExists(departmentId))`
    — ветка «нет»: Блок-4 процесс: `print("Ошибка: отдел"); return`
    — ветка «да» → Блок-5
Блок-5 условный: `if (!positionExists(positionId))`
    — ветка «нет»: Блок-6 процесс: `print("Ошибка: должность"); return`
    — ветка «да» → Блок-7
Блок-7 процесс: `worker = Worker(...); record = EmployeeRecord(worker, hireDate)`
Блок-8 процесс: `employees.add(record); registerEmployee(record)`
Блок-9 процесс: `pushUndo(removeEmployee(newId))`
Блок-10 вывод: `print("Работник добавлен")`
Блок-11 овал: `Финиш`

Алгоритм 3. Поиск сотрудников
Блок-1 овал: `Старт`
Блок-2 ввод: `filter = readFilters()`
Блок-3 процесс: `result.clear()`
Блок-4 процесс: `it = employees.begin()`
Блок-5 условный: `if (it == employees.end())`
    — ветка «да»: переход к Блок-9
    — ветка «нет» → Блок-6
Блок-6 условный: `if (matchesFilters(*it, filter))`
    — ветка «да»: Блок-7 процесс: `result.push_back(*it)`
    — ветка «нет»: минуем Блок-7
Блок-8 процесс: `++it` и возврат стрелкой к Блок-5
Блок-9 вывод: `printResult(result)`
Блок-10 овал: `Финиш`

Алгоритм 4. Отмена действия
Блок-1 овал: `Старт`
Блок-2 условный: `if (undoStack.empty())`
    — ветка «да»: Блок-3 процесс: `print("Нечего отменять")`; переход к Блок-7
    — ветка «нет» → Блок-4
Блок-4 процесс: `command = undoStack.pop()`
Блок-5 процесс: `undoInProgress = true`
Блок-6 процесс: `command.undo(); undoInProgress = false`
Блок-7 вывод: `print("Отменено: " + command.description)`
Блок-8 овал: `Финиш`
