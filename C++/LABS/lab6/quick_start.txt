Quick start (Lab 6)

Requirements:
- CMake 3.30+ and a C++20 compiler (clang++/g++).

Build:
1) From the lab6 directory: cmake -S . -B build
2) Build: cmake --build build

Run:
- Start the console app: ./build/laba
- Меню: добавить/показать/удалить/искать/сортировать/размер кольца.
- Дополнительно: пункты 7–10 записывают/читают кольцо в текстовый или бинарный файл (имя файла можно ввести вручную).
